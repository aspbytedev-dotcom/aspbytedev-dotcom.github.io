# Salamatutkan verkkosivusto

AspByten julkinen GitHub Pages -sivusto Salamatutka-sovellukselle.

Sivusto sisältää:

- suomen-, ruotsin- ja englanninkielisen tuote-esittelyn
- Free- ja Pro-versioiden ominaisuudet ja hinnat
- sovelluksen tietolähteet ja turvallisuushuomautuksen
- kolmikielisen tietosuojakäytännön
- AdMobin vaatiman `app-ads.txt`-tiedoston

Tavoiteosoitteet:

- https://aspbytedev-dotcom.github.io/
- https://aspbytedev-dotcom.github.io/privacy-policy.html
- https://aspbytedev-dotcom.github.io/app-ads.txt

Sivusto ei käytä analytiikkaa, evästeitä tai ulkoisia fontteja. Valittu kieli tallennetaan selaimen paikalliseen tallennustilaan.
